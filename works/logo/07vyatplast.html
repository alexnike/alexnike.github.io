Инакор

Логотип для спортивной игры

Инакор это увлекательная игра на основе спортивного ориентирования для взрослых и детей в городских парках.

Логотип проекта должен был соответствовать ряду критериев:

✦ Логотип должен быть персонажем
✦ Пригодный для модификаций
✦ Привязка к сути игры
✦ Минимум цветов
✦ Контрастность
✦ Возможность печати в одном цвете
✦ Простой и запоминающийся

В качестве основы для разработки персонажа мной был использован элемент карт — метка. Добавил руки, улыбку, глаза и персонаж готов.

Поскольку клиент хотел, чтобы логотип был пригоден для модификаций, сделал несколько альтернативных вариантов.

Логотип можно менять добавляя к основе любые элементы, наиболее подходящие к контексту события.

Подготовлено краткое руководство по использованию знака при размещении его на различных носителях.
